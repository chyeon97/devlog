## 블로그 제작

> 학습한 내용을 기록하기 위한 저장소

### 디렉토리 구조 및 설명

| 디렉토리명      | 설명                                                    |
|------------|-------------------------------------------------------|
| `contents`   | 블로그 포스트 관련 파일들을 저장                                    |
| `components` | React Components 저장                                   |
| `hooks`      | Custom Hooks 저장                                       |
| `pages`      | 페이지 역할을 하는 컴포넌트 저장                                    |
| `templates`  | 게시글 페이지와 같이 페이지의 역할을 하면서 같은 형식의 여러 콘텐츠를 보여주는 컴포넌트를 저장 |
